# Vita
Vita is a journaling tool and social network inspired by Github's activity heatmap.

## Stack
* Frontend: [Vue3](https://vuejs.org/)
* UI: Bootstrap via [Bootswatch](https://bootswatch.com/)
* Backend: [Express](https://expressjs.com/)
* DB: Postgres via [Supabase](https://supabase.com/)
* ORM: [Prisma](https://www.prisma.io/)
